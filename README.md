# Cabinet Calculator

Kitchen cabinet carcass calculator with drawer support — Angular SPA that optimizes board cutting layouts.

## Quick Start

```bash
npm install
npm start        # dev server at localhost:4200
npm test         # run tests (vitest)
npm run lint     # eslint
npm run format   # prettier
```

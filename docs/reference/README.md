## When to Create Documents

| Type | Trigger |
|------|---------|
| ADR | Nontrivial technology choice with rejected alternatives |
| Guide | Recurring procedure that takes >5 minutes |
| Manual | Library/tool with project-specific configuration |
| Research | Investigation with data to preserve |

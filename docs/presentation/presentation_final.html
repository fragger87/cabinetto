Cabinet Calculator

Kitchen cabinet carcass calculator with drawer support

Angular SPA · Guillotine bin packing · Two-stage parametric optimization

Problem

Designing European frameless kitchen cabinets requires computing optimal board cutting layouts that minimize sheet material waste.

Manual layout planning is time-consuming and suboptimal. Small changes in leg height or cabinet depth can save entire boards.

Solution

User Input → ProjectConfig → Stage 1: Leg Height Sweep (5mm steps) → Stage 2: Depth Sweep (5mm steps) → Guillotine Bin Packing (BSSF) → Visual Report + BOM

Two-stage parametric optimization finds the combination that uses the fewest boards with highest utilization.

Tech Stack

Angular 21 TypeScript 5.9 Vitest ESLint + Prettier GitHub Actions

No backend. No runtime dependencies beyond Angular core.
All computation in pure TypeScript services.

Architecture

LayerComponents
InputBoard spec, cabinet list, drawer config, polymorphic fields
StateSignal-based ProjectStateService
ComputeCuttingOptimizer, LegOptimizer, DepthOptimizer, BOM
ReportSVG cutting layouts, parts tables, cabinet viz, print
PersistencelocalStorage + JSON import/export

Key Feature: Drawer Support

SchemeDescription
EqualAll drawers same height
GraduatedTop shorter, bottom taller (1:1.5:2)
CustomUser-specified heights per drawer

15mm drawer boards optimized in a separate pass from 18mm carcass boards.
